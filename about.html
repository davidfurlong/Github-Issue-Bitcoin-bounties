About GitBounty

GitBounty is a platform for:
  • Crowdfunding Open Source software
  • Connecting developers with jobs
  • Encouraging Open Source contribution

John wants to use some open source code for his business but it lacks a feature or modification which he needs. But he doesn't have the knowledge to do it himself. Finding someone to do it for him would be time consuming, costly and they would probably not be the best suited person to do the work he needs.

John can simply add a new bounty to GitBounty, which will notify the developers of the Github project that completing the issue John has created will be rewarded with a certain amount of bitcoins. Additionally, John's bounty is posted as a comment to the issue so anyone subscribed to the issue on Github will be notified. Finally the bounty is displayed in a list on GitBounty where developers who want to contribute to open source software and get paid can find opportunities which match their skillsets.
